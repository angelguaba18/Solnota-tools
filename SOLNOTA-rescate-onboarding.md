# RESCATE SOLNOTA — Proceso de Onboarding (paso a paso)

**Qué es:** el flujo repetible para meter a CUALQUIER artista a SOLNOTA y rescatar su dinero. Mismo proceso para todos.
**Meta:** del "no sé nada de esto" al "ya estoy cobrando" sin que el artista se pierda.

---

## FASE 0 — Entrada (10 min)

- [ ] Artista firma el **Acuerdo 1 página** (entiende) + **Contrato de administración** (legal).
- [ ] Crear carpeta del artista: `/Artistas/[Nombre]/` con subcarpetas: `splits/`, `pruebas-DSP/`, `contratos/`.
- [ ] Anotar: nombre legal, p/k/a, email, país.

---

## FASE 1 — Diagnóstico / Escaneo (el gancho)

- [ ] Listar TODO el catálogo con **ISRC** (Apple/Spotify/distribuidora).
- [ ] Por cada obra marcar: ¿registrada en MLC? ¿en PRO? ¿tiene splits? ¿unclaimed?
- [ ] Sacar el número impacto: *"tienes X canciones, Y sin cobrar."* → esto vende.

---

## FASE 2 — Conexión (afiliaciones)

- [ ] **PRO:** afiliar a BMI o ASCAP (writer) → sacar **IPI**. Gratis.
- [ ] **MLC:** abrir cuenta songwriter. Gratis.
- [ ] **SoundExchange:** si suena en radio digital/Pandora.
- [ ] **Publisher:** obras bajo Sol Music Group Publishing (BMI) o Sol Latino Music (ASCAP) según el PRO del artista.
- [ ] **Banco:** cargar datos de pago en cada plataforma (para que el dinero caiga).

---

## FASE 3 — Splits (reconstruir si faltan)

- [ ] Obra de 1 escritor → 100%, registrar directo.
- [ ] Obra con varios → créditos DSP + partes iguales + confirmación por mensaje.
- [ ] Lo dudoso → lista aparte (NO registrar hasta confirmar). Evita overclaim.
- [ ] Guardar cada prueba (screenshot + fecha) en `pruebas-DSP/`.

---

## FASE 4 — Registro y Reclamo

- [ ] Registrar obras nuevas en MLC + PRO (CWR si es en lote).
- [ ] **Claiming Tool (MLC)** → reclamar shares unclaimed.
- [ ] **Matching Tool (MLC)** → conectar grabaciones sueltas.
- [ ] Reclamar retroactivo recuperable (MLC desde 2021, SoundExchange 3 años).

---

## FASE 5 — Mantenimiento

- [ ] Resolver cualquier **overclaim** (ver protocolo anti-bloqueo).
- [ ] Reporte al artista: cuánto entró y de dónde.
- [ ] Regla nueva: **split sheet ANTES de soltar cada canción** (nunca más perder uno).

---

## Checklist rápido (versión corta)

1. Firma → 2. Escaneo ISRC → 3. Afiliar (PRO+MLC+IPI) → 4. Splits → 5. Registrar+Reclamar → 6. Reportar
